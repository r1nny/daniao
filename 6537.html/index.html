    关于ssh登录

    请用xshell生成公有密钥和私有密钥，然后把把后缀为pub的上传到【添加 SSH 密钥】中，如图：

    注意事项：SSH 密钥填写千万要注意！这个密钥必须填！还必须得对！否则开的云服务器无法登录!!!

    修改为密码登录

    登录到云服务器，如果是Centos系统直接使用命令sudo -i 就可以切换到root权限，如果你想要使用root登录，请使用以下命令（密码是daniao，你可以替换为其它的）：

    echo root:daniao |sudo chpasswd root
sudo sed -i 's/^#\?PermitRootLogin.*/PermitRootLogin yes/g' /etc/ssh/sshd_config;
sudo sed -i 's/^#\?PasswordAuthentication.*/PasswordAuthentication yes/g' /etc/ssh/sshd_config;
sudo service sshd restart

    如何重装系统

    #Debian/Ubuntu:
apt-get update
