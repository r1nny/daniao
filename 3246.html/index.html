    谨慎入坑-Linode后台”Linode Backup Auto-enrollment”设置问题

    2018年9月17日21:34:51主机教程26,1411172字阅读模式

    我们这两天登录linode后台的时候，都会看到Linode有"Linode Backup Auto-enrollment"这样的提示信息，大鸟乍一看吓了一跳以为出现什么违规信息了。然后仔细看了之后发现并不是这样的，而是用来推荐他们的自动备份服务默认设置的。

    linode推送的英文全文如下：

    Linode Backup Auto-enrollment

    We recently added an account setting for automatic Linode Backup enrollment. With this setting set to "AUTOMATIC", all new Linodes will be created with the Linode Backup Service enabled. To enable backups on all existing Linodes, use the second link below.

    • Edit Account Settings (future Linodes)
    • Enable Backups for all Linodes (existing Linodes)

     

    意思就是说，我们可以设置一键将所有的VPS都设置自动备份服务，当然是收费的，如果我们只是在测试项目，那完全没必要搞什么自动备份，所以这里选择一定要谨慎，不要不小心点了全部自动备份，自动备份是收费的价格不便宜。

    很多人可能不清楚就去一键启动，所以我们需要看清楚，如果不需要，那就点击上面的一个"Edit Account Settings (future Linodes)"。让我们看看如何来设置不自动备份。

    一、Edit Account Settings

    我们选择第一个修改帐户设置（未来的Linodes）如果你不知道选择哪个，我们看图：

    我们选择标注红框的选项，来设置linode。然我们我到下一步。

    二、Account Settings

    账号设置这里如大鸟这样设置就可以了。不会的就看图：

    设好好了，我们选择“Save”来保存！这样设置就可以不掉进自动备份的坑里面了。我们可以看这里来确定是否设置没问题了。

    Backups这里没有开启自动备份，所以这样设置是没问题的，不会掉坑。

    当然，如果我们需要设置一键全部设置备份，点击"Enable Backups for all Linodes (existing Linodes)"。

    我们可以看到一键可以启动账户中的所有的Linode机器，当然是收费的。大鸟这只有一台所以收费便宜点。如果很多台机器，那这里的收费还是很吓人的。

    三、总结

    大鸟很奇怪Linode后台有这个备份一键设置提示，以前都是自己自由选择的，linode这样搞就有点乱来了，所以我们需要小心不要乱点击启动。其次，如果我们需要设置备份那就设置，如果不需要，就检查设置手动再开启，不要设置自动。不过，就算是开启了自动备份，也是可以关闭的，也不用太担心。
