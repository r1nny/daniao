一：永久VPS篇

这里整理一些永久免费的vps，如果有遗漏，欢迎留言补充！！

1、甲骨文

1）特点：

顶级羊毛，无数mjj为之疯狂。特点，永久免费，数据中心多达24个，之前好用的首尔和东京因为众多mjj的涌入导致线路拥堵不堪！此外ARM机型可自由分配 4个OCPU，24 GB 内存，最大200GB空间容量和4GB的带宽。

2）线路选择：

电信：首尔，东京，春川，迪拜，圣何塞，凤凰城！！

移动和联通随意选吧！！！

3）申请教程：

如果你还没有甲骨文，可以看之前的申请教程：甲骨文申请

4）申请后你可以需要刷机脚本

arm刷机js脚本：Oraclecloud甲骨文 – 免费VPS获取自动脚本代码

Terraform刷机：Oracle甲骨文 ARM VPS（VM.Standard.A1.Flex）自动抢购脚本代码

Evolution Host vps是可以免费申请的，无需信用卡，也不需要注册，只需要简单发送一封包含自己网站地址的邮件，Evolution Host需要评估之后会给你开通一台永久免费的vps。

注意，你的网站需要流量不错，门槛稍微有点高！！

6、digitalocean

一下简称do，do的一大特色就是各种送码和优惠！

1）do有很多分享链接和推广链接，使用这些链接注册后双方都能得到一些奖励，目前官网的活动，通过 https://m.do.co/c/b2289139c81a 注册会被赠送 100 美金，但有效期只有两个月，但有效期时间太短，并不推荐使用。

2）推荐用无痕浏览器注册，不使用任何推广链接注册，这样注册后可以使用优惠码，目前金额最大的优惠码属于 GitHub 学生礼包 中的促销码，同样 100 美金，有效期一年。

GitHub 学生开发工具包申请：https://www.daniao.org/8860.html

3）这里简单说下do开子号吃码！

  • 步骤1 创建主号 验证信用卡
  • 步骤2 创建团队勾选信用卡 将主号转换为团队我没勾
  • 步骤3 邀请其他邮箱 加入团队 然后主号把子号提升为所有者
  • 步骤4 在主号邮箱里点击同意子号为所有者、
  • 步骤5 主号退出团队 子号登录，提示要绑信用卡或者PP不用管他 右上角切换为团队号 然后吃码子即可

9、AWS

asw里比较香的是免费套餐里的亚马逊云（EC2），可以免费使用12个月，可选择美国东、西节点，日本东京以及新加坡等地多个节点。特别适合囊中羞涩和小白用户折腾练手，速度也不错。

这里提醒下大家，这个免费VPS他是有流量限制的，限制共 15GB 的带宽传出量。你可以使用亚马逊云（EC2）来建立自己的网站。亚马逊毕竟是大公司，服务器的稳定性还是相当不错的。

13、freenom

大名鼎鼎的免费域名服务商，提供.tk、.ml、.ga、.cf、.gq等众多后缀的免费域名！

14、eu.org

简单介绍下“EU.org, free domain names since 1996”。eu.org是欧盟组织下面的域名，EU代表欧盟，Paul Mockapetris在1996年的9-10月份创建了这个域名的DNS服务器。现在对个人和组织（没见提是不是非盈利性）是免费注册的。目前已经被谷歌，cf等一些大公司认可为顶级域名~~~~

申请教程：https://www.daniao.org/13789.html

15、pp.ua

.ua 是乌克兰的国家域名后缀，pp.ua中的PP是个人、私人的缩写(Private Person)。pp.ua是乌克兰官方为个人提供的免费二级域名后缀，任何人都可以申请。pp.ua由乌克兰官方运行，稳定性有保障。

申请教程：https://www.daniao.org/13806.html
