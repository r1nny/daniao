    Directory Lister-最简洁目录列表程序，优秀的在线文件管理器

    2019年4月8日16:11:18在线应用528,3461165字阅读模式

    Directory Lister是一款简洁的基于PHP的文件目录列表程序，程序安装非常简单，直接上传文件然后打开浏览器就可以访问了。

    Directory Lister是开源代码，没有广告，目前的最新版已经很好的支持中文了。这篇文章，我们来看看，如何安装使用Directory Lister！

    如果不喜欢Directory Lister的极简风格，可以看看大鸟之前的文章：

    1. h5ai-轻便又美观目录列表程序支持在线预览文本、图片、音频、视频等
    2. 宝塔面板-安装功能强大的文件管理器FileRun的详细教程

    一：Directory Lister介绍

    Directory Lister支持PHP 5.3或更高版本。您可以通过运行来验证您的PHP版本php --version

    官网：http://www.directorylister.com/

    演示：https://demo.directorylister.com/

    项目地址：https://github.com/DirectoryLister/DirectoryLister

    下载zip：https://api.github.com/repos/DirectoryLister/DirectoryLister/zipball/2.7.1

    下载tar.gz：https://api.github.com/repos/DirectoryLister/DirectoryLister/tarball/2.7.1

    哪里可以获得Directory Lister新闻和更新？目前，获取更新的最佳方式是在Twitter上关注@DirectoryLister。

    二：安装

    下载之后即可上传到自己网站根目录安装，文件大小只有17.8k，需要修改的地方就是：

    resources/default.config.php 重命名为 resources/config.php。

    上传设置好之后，我们就可以用域名打开这个在线列表程序了，无需安装，开箱即食！

    三：使用

    Directory Lister最新版的已经很好地支持中文了，\resources\themes\bootstrap里面有主题，你可以根据自己的需要来修改CSS等。

    可以新建文件夹上传到里面，或者直接放在根目录，如图：

    大鸟新建了daniao目录然后把Directory Lister上传到里面，这样就可以下载了。用网址打开看看效果。

     

    四：总结

    Directory Lister是一个基于PHP的简单脚本，可以让您列出目录及其所有子目录的内容，只需将Directory Lister下载并安装到任何Web目录，即可立即访问该目录下的所有文件和子目录。

    不过对大鸟来说，界面风格简洁流畅，没有广告，没有花哨，没有可怕的各种版权声明，对大鸟来说足够使用了。
