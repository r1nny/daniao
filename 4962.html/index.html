    宝塔面板Docker安装UnblockNeteaseMusic – 解锁网易云音乐变灰歌曲

    2019年5月4日16:52:34宝塔面板2255,6731804字阅读模式

    UnblockNeteaseMusic：解锁网易云音乐客户端变灰歌曲，支持Windows、MacOS、安卓、IOS等设备，特性如下：

    • 使用网易云旧链 / QQ / 虾米 / 百度 / 酷狗 / 酷我 / 咕咪 / JOOX 音源替换变灰歌曲链接 (默认仅启用前四)
    • 为请求增加 X-Real-IP 参数解锁海外限制，支持指定网易云服务器 IP，支持设置上游 HTTP / HTTPS 代--理
    • 完整的流量代理功能 (HTTP / HTTPS)，可直接作为系统代理 (同时支持 PAC)

    我看看，什么是变灰的歌曲，如图：

    这篇文章大鸟用宝塔面板的Docker来安装UnblockNeteaseMusic，也算是一篇Docker教程贴。

    一：安装

    自备一台vps，配置不用太高，512内存1核cpu即可，安装好宝塔面板备用。这个就不多赘述了。

    github：https://github.com/nondanee/UnblockNeteaseMusic

    1.1安装docker

    宝塔面板内置node.js管理器所以不用安装了，我们只安装docker即可，

    1.2添加镜像

    点开docker的设置找到镜像管理点击获取镜像，我们输入镜像地址：

    nondanee/unblockneteasemusic

    输入好之后，点击获取镜像即可，丢一张图看看：

     

    1.3：创建容器

    点击容器列表→创建容器，这样设置，如图：

    容器端口4567，自己可以修改，容器的4567端口映射到主机的8080端口，最后执行命令处要填写：

    node app.js

    最后提交，执行命令一定要填写，不填写是不能创建容器的，记住自己的4567端口一会要用。

    1.4命令创建容器

    额，这个图形过程太繁琐了，如果你看不懂，我们一行命令就可以解决:

    docker run --name yunmusic -d -p 4567:8080 nondanee/unblockneteasemusic

    我们用一个命令来看看容器状态：

    命令如下：

    docker ps

    这样我们Docker安装UnblockNeteaseMusic就完成了，下面开始设置。

    二：Winodws端设置

    2.1修改Hosts

    Windows下找到Hosts路径：

    Windows系统：位于C:\Windows\System32\drivers\etc\hosts

    向 hosts 文件添加两条规则：

    <Server IP> music.163.com
<Server IP> interface.music.163.com

    2.2客户端设置

    大鸟说的是Windows下的网易云音乐客户端，设置方法：

    • 网易云客户端设置 > 工具 > 自定义代理 > HTTP代理
    • 这里只需要填上你的服务器ip和端口，保存即可。

    丢一张图：

    设置好之后，我们可以重启客户端，看看效果，如图：

    三：Android端设置

    如果我们用的是Android手机，设置也很简单，一般安卓手机设置代理类型的时候分2种，手动代理和自动代理，有的手机可能只有手动代理，大致设置如下：

    • WLAN > 修改网络 > 高级选项 > 代理
    • 手动代理 > 只需要填上你的服务器IP和端口就可以了。
    • 自动代理 > 填上你的P-A-C地址就行了，地址：http://<Server Name:PORT>/proxy.pac，修改为你的ip、运行端口。 如果你发现设置后不能使用或者出现其它网络问题，带了-s参数的不妨去掉后再试试。

    丢一张图看看:

    四：总结

    我用电脑端点开速度还是挺快的，手机端有点缓慢，可以稍微等等，点亮之后，音乐下载也没有限制，可以下载到本地。

    有一点要补充，忘了说了，以上说的端口，我们要去宝塔面板→安全→防火墙里面放行用到的端口。
