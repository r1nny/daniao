    宝塔面板Python项目管理器安装tg_fa_bot

    2020年6月29日23:52:41宝塔面板2615,0571343字阅读模式

    tg_faka_bot是一个TG发卡机器人，此机器人基于Python开发，大鸟在Python 3.7.2测试通过。

    这篇文章就来说下，如何用宝塔面板搭建tg_faka_bot。

    1、简介

    项目：https://github.com/lulafun/tg_faka_bot

    2、准备

    需要的东西很简单，如下：

    1）宝塔面板（宝塔服务器面板，一键全能部署及管理，送你3188元礼包，点我领取https://www.bt.cn/?invite_code=MV9ub2NxdmI=）

    2）准备bot

    3）Python项目管理器（软件商店→宝塔插件）

    3、bot创建以及获取admin_id

    这里简单说下如何在Telegram创建一个bot。至于如何打开Telegram不在赘述。

    1）直接给 BotFather（https://t.me/botfather）发送 /newbot 指令创建机器人，第一次给你的机器人起一个名字，第二次输入（ It must end in `bot`. Like this, for example: TetrisBot or tetris_bot.）具体看图：

    此处为隐藏的内容！
    发表评论并刷新，方可查看

    4、部署

    1）下载项目

    git clone https://github.com/lulafun/tg_faka_bot.git

    2）修改config.py文件

    编辑config.py文件，根据注释配置参数。

    # 这是开源版本

# 易支付API地址, 末尾需要包含/
API = 'https://home.bakbak.cn/'
# 商户ID
ID = 12123
# 商户密钥
KEY = 'DZNEk2dQzKD10NCX0RwR'

# 支付成功跳转地址
JUMP_URL = "https://kangle.bakbak.cn/paysuccess.html"
# 支付超时时间(秒)
PAY_TIMEOUT = 300
# BOT API
TOKEN = '1297987814:AAG4kO2l-6wuPwoPMF7imHOJtpYAI'
# ADMIN ID
ADMIN_ID = [7442874233]

# 管理员命令
ADMIN_COMMAND_START = 'iadmin'
ADMIN_COMMAND_QUIT = 'icancel'

    需要修改的地方很多，基本你都能看出来需要修改什么，这里的BOT API和ADMIN ID就是你上面获取的。

    3）安装

    打开Python项目管理器添加项目，具体看图：

    点击确定后项目开始运行。

    5、测试效果

    1）输入命令

    和你刚刚新建的机器人对话，输入命令(/start)即开始工作。

    输入命令（/iadmin）即是管理员管理界面，可以添加分类，商品等，如图：

    2）看视频

    6、最后

    安装不难，但是很多东西需要你自己设置了。这个算是神器了，因为你有了它，再也不用怕被DDCC了。

    该机器人使用sqlite3作为数据库，轻量、便于备份。其他的东西都自己折腾把。
